			font-family

Softwares Installation
**********************
1) download and install browser
	
		Google Chrome

		Mozilla


2) download and install Visual Studio Code

		website : https://code.visualstudio.com/docs/?dv=win

		file    : VSCodeUserSetup-x64-1.67.2.exe


3) install LiveServer plugin
	
		LiveServer plugin detectes the code changes

		LiveServer plugin will give code changes information to browser

		Browser will reload automatically with code changes


		Click the Extensions ==> Live Server (Search) ==> Install



4) Enable AutoSave Mode in VSCode

	click settings icon ==> settings ==> Auto Save ==> after Delay


5) make google chrome as default browser


	click the setting icon (right top corner of browser) ==> settings 
   ===> default browser
